# 객체와 클래스 #

```
class Person():
    def __init__(self):
        pass
```

__init__() > 클래스의 정의로부터 객체를 초기화시키는 메소드
self > 객체 자신을 가르킴

def __init__(self, name):
 self.name = name
 
 > name 메개변수를 초기화 메서드에 추가할 수 있음
 
 코드의 동작 형태
 > 
 1. Person 클래스의 정의를 찾는다.
 2. 새 객체를 메모리에 초기화(생성).
 3. 객체의 __init__ 메서드를 호출, 생성된 객체를 self에 전달, 인자를 name에 전달
 4. 객체에 name 값을 저장
 5. 새로운 객체를 반환
